# v0.1.0 Release Notes

Initial open-source release of Webchat Bridge.

Highlights:
- Gateway API + WebSocket relay
- Embeddable widget (`/widget.js`)
- Discord thread bridge
- WordPress plugin with admin dashboard
- Email-based ticket escalation
- Session tracking + GeoIP map

Notes:
- AI chat features available via OpenAI integration.
- WordPress plugin included for easy integration.
